# Install steps

## Step 1
Go https://github.com/ouster-lidar/ouster-ros/tree/ros2 and follow the install steps

## Step 2
Go to "wired settings" <br/>
Click on the settings in the 'Wired' section <br/>
Go to the IPv4 tab <br/>
Click on Link-Local Only <br/>

## Step 3
Make a ros2 workspace on the drive then clone into this folder

## Step 4
Find the serial number of the lidar (Provided in the lidar page)

## Step 5
Access the lidar webserver using the link provided in the lidar page
[Serial number].local

## Step 6
Ensure firmware is up to date

## Step 7
Launch the lidar driver following the steps in the ouster-ros readme
